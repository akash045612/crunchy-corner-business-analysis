# Crunchy Corner Business Analysis

## Project Overview
This project analyzes 5 years of business data (Actual vs Budget) to evaluate revenue, cost, and profitability.

## Tools Used
- Power BI
- DAX
- Data Modeling

## Key KPIs
- Net Revenue (NR)
- Gross Profit (GP)
- EBITDA
- Net Profit (PAT)
- Raw Material Cost %

## Key Insights
- Revenue remained stable around 110–115M
- Protein Pack is the top-performing category
- UP and Maharashtra contribute highest revenue
- RM Cost % is stable around 49–50%
- Some categories show lower GP% and need improvement

## Business Impact
This dashboard helps in identifying high-performing areas and supports data-driven decision making.
